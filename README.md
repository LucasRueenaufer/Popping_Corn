# Popping Corn

Popping Corn is a small pet project, mainly concerned with the statistics of popcorn popping. It includes audio processing to extract information about popcorn from an audio recording of its preparation and then runs a short analysis on that data.

## Installation

This package is not really meant for installation, although you are welcome to scoure the code for anything that might be useful to you.

## Contributing
As this is a small pet project, I do not really expect contributions. If you got any tips or modifications, pull requests are still welcome :)
